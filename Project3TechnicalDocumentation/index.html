Introduction to Java

Java is an object-oriented programming language developed by James Gosling and his team at Sun Microsystems in 1995. Today, it is managed by Oracle Corporation.

One of Java's most important slogans is "Write Once, Run Anywhere," which emphasizes its platform-independent nature.


// First Java application
public class HelloWorld { 
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

When Java code is compiled, it creates an intermediate code called bytecode, which can run on any operating system using the JVM (Java Virtual Machine).

Java is widely used in various domains including Android applications, web services, enterprise applications, and big data applications.


// Checking Java version
java -version

Java Fundamentals

In Java, every program is written within a class structure, and each application starts with the main() method. Java is statically typed, meaning every variable has a data type.

The primitive data types in Java are: byte, short, int, long, float, double, boolean, and char. Besides these, you can use reference types such as String.


// Variable declaration examples
int number = 10; 
double pi = 3.14;
char character = 'A';
boolean isTrue = true;
String text = "Java Programming";

Control structures in Java are similar to other languages. It includes if-else, switch, for, while, and do-while statements.


// Loop example
for (int i = 0; i < 5; i++) { 
    System.out.println("Number: " + i);
}

Methods in Java are defined within classes and can work with parameters. Methods can have a return type or be declared void if they don't return any value.

Object-Oriented Programming

Java is a fully object-oriented language that supports all the features of this paradigm. Classes and objects are the fundamental building blocks of Java.

There are four main principles of object-oriented programming:

  • Encapsulation: The bundling of data and methods within a class
  • Inheritance: The ability of a class to inherit properties from another class
  • Polymorphism: The ability of a method to behave differently in different contexts
  • Abstraction: Simplifying complex systems by modeling classes
  • Interfaces: Defining contracts for classes to implement

In Java, inheritance is achieved using the "extends" keyword. A class can inherit from only one class (single inheritance) but can implement multiple interfaces.

Interfaces are implemented using the "implements" keyword and require classes to include certain methods.

Exception Handling

Java uses exception handling mechanisms to manage errors that might occur during runtime. Exceptions allow you to catch and properly handle potential errors in your code.


// Try-catch exception handling
try { 
    int result = 10 / 0;  // Throws ArithmeticException
} catch (ArithmeticException e) {
    System.out.println("Division by zero error: " + e.getMessage());
} finally {
    System.out.println("This block always executes");
}

Java Collections

The Java Collections Framework is a set of classes and interfaces used to store and process data. The most common collection types include ArrayList, LinkedList, HashSet, TreeSet, HashMap, and TreeMap.


import java.util.ArrayList;

public class CollectionExample {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<>();
        list.add("Java");
        list.add("Python");
        list.add("C++");
        
        for (String language : list) {
            System.out.println(language);
        }
    }
}
